Champak Search

A simple personal search engine that can be used from the Chrome address bar.

Chrome URL format: https://search.learnwithchampak.live/?q=%s Press Enter to search

Install in Chrome as a search engine

In Chrome, open chrome://settings/searchEngines. Under Site search, click Add.

Use these values:

Search engine: Champak Search

Shortcut: cs

URL with %s in place of query: https://search.learnwithchampak.live/?q=%s

Then type cs python in the Chrome address bar.
